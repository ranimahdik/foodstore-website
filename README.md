# 🍔 Food Store Website

## 📌 Project Description
This is a mini project developed for Web Technology subject.  
It is a Food Store Website where users can explore food items, view products, and interact with the interface.

---

## 🚀 Technologies Used
- HTML  
- CSS  
- JavaScript  
- JPG Images  

---

## ✨ Features
- 🏠 Homepage with attractive UI  
- 🛒 Product display  
- 🔐 Login & Register pages  
- 📞 Contact page  
- 🌙 Dark mode toggle  
- 🎨 Responsive design

## 🎯 Objective
The objective of this project is to design and develop a user-friendly Food Store Website using HTML, CSS, and JavaScript.  
It helps in understanding web development concepts, responsive design, and UI creation.
